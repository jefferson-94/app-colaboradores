## Requisitos
* Node.js 20 ou superior
* Expo

## Como rodar o projeto baixado
Instalar todas as dependencias indicadas no package.json
'''
npm install
'''

## Inciar o projeto
```
npx expo start
```

## Sequencia para criar o projeto
'''
npx create-expo-app . --template
'''

## Execultar o projeto
'''
npx expo start
'''
